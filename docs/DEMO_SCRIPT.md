# Demo script (~3 minutes)

## Setup

- Pre-fill a brief: **“Q3 roadmap update”**, 2-minute limit, 3 takeaways filled in  
- Have one **pre-recorded** 60s clip as backup if live mic fails  

## Flow

1. **Intro (20s)** — “Speech Buddy helps you practice talks and get Toastmasters-style feedback aligned to what you want the audience to remember.”

2. **Brief (40s)** — Show format, title, description, three takeaways, time limit, watch-for checkboxes → Continue.

3. **Practice (30s)** — Record 30–45s OR upload backup file → Analyze.

4. **Feedback (60s)** — Walk through:
   - Timing vs limit  
   - Filler count  
   - Content / Delivery / Language scores  
   - Three takeaway alignment items  
   - Transcript (expand)

5. **Close (30s)** — “Practice again” keeps the same brief; mention backend STT + LLM for production.

## Fallback

If API is down: demo still works in mock mode — say you’re showing UI while API deploys.
